## Safe to use fonts:

* Arial,
* Courier New, Courier,
* Garamond,
* Georgia,
* Lucida Sans, Lucida Grande, Lucida,
* Palatino Linotype,
* Tahoma,
* Times New Roman, Times,
* Trebuchet,
* Verdana.

## Hosting your fonts:
``` css
@font-face {
font-family: "Myfont";
src: local("Myfont"), url("my_font.woff") ,
format("woff");
}
```
